Partie 1. Comparaison

library(ggplot2)
library(data.table)

Comparison des caractéristiques des fleurs

Dans la série de données iris, on peut comparer les différentes valeurs des 4 variables des fleurs. L’objectif est de montrer les valeurs d’une ou de plusieurs variables en fonction des trois espèces.

Bar plot

data=data.table(iris)
res=data[,.(m=mean(Sepal.Length)),by=Species]
barplot(res$m,names.arg=res$Species,
        main="Longueurs moyennes des sépales en fonction des espèces")

  • Ajouter les volatilités de part et d’autre des moyennes
  • Ajouter quelques quantiles
  • Afficher plusieurs variables

Boxplot

Le boxplot permet d’afficher quelques statistiques simples des variables. Identifier quelles sont les valeurs affichées.

p <- ggplot(iris, aes(Species, Sepal.Length))+ geom_boxplot()
p

Violin plot

Le violin plot permet de donner la forme des densités.

p <- ggplot(iris, aes(Species, Sepal.Length))
p + geom_violin()

Rajouter les les statistiques de bases (moyenne, volatilité, quantiles) sur le violin plot.

Tabplot

Pour ce graphique, utilisez le package tabplot.

library(tabplot)
tableplot(iris, sortCol="Species")

library(MASS)
