  • Exemples sur les 3 premières séances
  • Discussions des projets
  • Visualisation des algorithmes de machine learning

2 Machine learning

2.1 Comprénsion des algorithmes

La visualisation permet d’abord de mieux comprendre le fonctionnement d’un algorithme. Dans les exemples suivants, il n’y a pas de graphique fait, il est possible de les créer pour une meilleure compréhension.

  • Naive Bayes
  • kNN

Comme projet, vous pouvez choisir un algorithme à expliquer de façon visuelle.

2.2 Utilisation des algorithmes

Pour présenter les résultats de prédiction des algorithmes, on peut présenter les résultats de façon visuelle.

  • Détection de spams
  • SVD et PCA
  • FFT (ce n’est pas un algorithme de machine learning à proprement parler. Mais son utilisation dans le traitement d’images a un effet visuel marquant.)

2.3 Explication des algorithmes

Avec Python, cette comparaison d’algorithmes visuelle est percutante. Avec R, un début de travail est commencé avec ce comparatif. Il est possible d’approfondir en ajoutant d’autres séries de données, des coefficient AUC, plusieurs niveaux de probabilités, etc.
